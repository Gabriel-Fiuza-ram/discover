Tecnologias utilizadas no projeto:

- JavaScript
- Git e Github

<br>

## 💻 Projeto

Um agregador de links para usar como cartão de visitas online.

<br>

## ⚒️ Features

- [x] Adicionar café no carinho
- [x] Remove café do carinho
- [x] CEP automático
- [x] Método de pagamento
- [x] Soma de todos os café

<br>
